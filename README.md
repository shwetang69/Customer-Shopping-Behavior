# Customer-Shopping-Behavior
Analyzed customer shopping patterns using Python, SQL, and Power BI to uncover key insights into purchasing behavior. Developed interactive dashboards and data-driven recommendations to support business decision-making.

Project Overview:
This project analyzes customer shopping behavior to identify purchasing patterns, trends, and insights that can help businesses improve sales and customer experience.

Tools & Technologies:
Python – Data cleaning, analysis, and visualization
SQL – Data extraction and querying from databases
Power BI – Interactive dashboards and reports.

Steps Performed:
Data Extraction using SQL queries from the database.
Data Cleaning and preprocessing using Python (handling missing values, duplicates, etc.).
Exploratory Data Analysis (EDA) to identify trends and patterns.
Visualizations and dashboards created in Power BI.
Insights and recommendations shared for business decision-making.

Outcome:
Identified high-value customers and their shopping preferences.
Highlighted seasonal trends and product demand patterns.
Created a dashboard for management to track sales and customer behavior in real-time.
